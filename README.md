# Customer-Churn-Analysis

🚀 How to Run the Analysis
Install DuckDB.
Download the Olist dataset from the Kaggle link above.
Ensure that the data folder is a directory below the database file.
Run CMD in the directory line where the folder is located
Type duckdb 'project-name'.db -ui
It will then move you to Duckdb to reproduce all KPIs and views. Ensure that the 'Environment Setup' section is executed first to ingest the CSVs into tables.
